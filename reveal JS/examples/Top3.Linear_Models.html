Linear Models:

  • When we say that $y$ is a linear function of $x$, we mean that the graph of the function is a line.
  • All of you are familiar with the idea of a linear model from learning the equation of a line, which is \begin{equation}\label{eq:line} Y=m X+b\tag{1} \end{equation} where $m$ is the slope of the line and $b$ is the $Y$-intercept.
    1. $y$ is the response variable (it responds to changes in $x$ )
    2. $x$ is the predictor variabl
    3. $m$ represents the estimated change in the response $y$ for a one-unit change in the predictor $x$
    4. $b$ is the value of the response $y$ when the predictor $x=0$

Linear Models:

  • It is useful to think of equation $(\ref{eq:line})$ as a function that maps values of $X$ to values of $Y$.
