  1. ស្ថិតិ

សេចក្ដីផ្ដើម

In real life, we work with data that are affected by randomness, and we need to extract information and draw conclusions from the data. The randomness might come from a variety of sources. Here are two examples of such situations:

  • Suppose we have a population with some unknown parameter(s). For example: $N(\mu,\sigma^2)$, where $\mu$ and $\sigma$ are parameters.
  • We need to draw conclusions (make inferences) about the unknown parameters.
  • We select samples, compute some statistics, and make inferences about the unknown parameters based on the sampling distributions of the statistics.

Classical Methods of Estimation (និយមន័យ)

  1. Point Estimation. A point estimate of some population parameter \(\theta\) is a single value \(\hat{\theta}\) of a statistic \(\hat{\Theta}\). For example, the value \(\bar{x}\) of the statistic \(\overline{X}\) computed from a sample of size $n$ is a point estimate of the population mean \(\mu\).
  2. Interval Estimation (Confidence Interval = C.I). An interval estimate of some population parameter \(\theta\) is an interval of the form \(\left(\hat{\theta}_{L}, \hat{\theta}_{U}\right),\) i.e, \(\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}\). This interval contains the true value of \(\theta\) with probability \(\mathrm{P}\left(\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}\right)=1-\alpha\). We call
    • $\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}$ is called a $(1-\alpha)100\%$ confidence interval (C.I.) for $\theta$.
    • $\theta_{L}$ is lower confidence limit, and $\theta_{U}$ is upper confidence limit
    • $1-\alpha$ is called the confidence coefficient
    • $\alpha= 0.1, 0.05, 0.025, 0.01$, $(0\lt \alpha\lt 1)$, i.e. ($\alpha = 1-\text{ level of confidence}$)


For example, you might have seen a problem like this: Let $X$ be a normal random variable with mean $\mu =100$ and variance $\sigma^2=15$. Find the probability that $X \gt 110$.

In real life, we might not know the distribution of $X$, so we need to collect data, and from the data we should conclude whether $X$ has a normal distribution or not.

១.  ចន្លោះជឿជាក់ (Confidence Intervals)

It is noted that we cannot be certain that an interval contains the true but unknown population parameter since only a sample from the full population is used to compute both the point estimate and, the interval estimate! A confidence interval is constructed so that there is high confidence that it does contain the true but unknown population parameter. Generally, a $100(1 - \alpha)\%$ confidence interval (CI) equals

$ \text{point estimate }\pm \text{ reliability coefficient } \times \text{ s.e.(parameter)} $
where
  • $\alpha$ is the level of significance between zero and one;
  • $1-\alpha$ is a value called the "confidence coefficient";
  • $100(1 - \alpha)\%$ is the confidence level;
  • parameter estimate is a value for the point estimate such as for the sample mean, $\overline{x}$, or for the population proportion, $\widehat{p}$;
  • reliability coefficient is a probability point obtained from an appropriate table as dictated by, for example, $z_{\alpha}$ or $t_{\frac{\alpha}{2},n-1}$;
  • and s.e.(parameter) read standard error of the parameter, measures the closeness of the point estimate to the true population parameter i.e. it measures the precision of an estimate in getting the parameter.


2. Single Sample - Estimation of the Mean ($\mu$)

យើងសន្មតជាទូទៅថា ទិន្នន័យជាសំណាក គឺបានមកពីប៉ូពុយឡាយស្យុងធម្មតាមួយ។

Recall: $$ \begin{array}{l} E(\overline{X})=\mu_{\overline{X}}=\mu; \quad \operatorname{Var}\big(\overline{X}\big)=\sigma_{\overline{X}}^{2}=\frac{\sigma^{2}}{n}; \quad \overline{X} \sim N\left(\mu, \frac{\sigma}{\sqrt{n}}\right) \\ Z=\frac{\overline{X}-\mu}{\sigma / \sqrt{n}} \sim \mathrm{N}(0,1) \text{ for } \left(\sigma^{2} \text { is known }\right); \quad T=\frac{\overline{X}-\mu}{S / \sqrt{n}} \sim \mathrm{t}(n-1) \text{ for } \left(\sigma^{2} \text { is unknown }\right) \end{array} $$ ក្នុងករណីនេះ យើងកំណត់តម្លៃសំណាករបាយអថេរចៃដន្យ $\overline{X}$ ដើម្បីធ្វើសេចក្ដីសន្និដ្ឋានរកតម្លៃប៉ារ៉ាម៉ែត្រ $\mu$ ។

  • Point Estimation of $\mu$. The sample mean \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) is a "good" point estimate for \(\mu\).
  • Interval Estimation (Confidence Interval) of $\mu$.
  1. First Case: $\sigma^2$ is known. If \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) is the sample mean of a random sample of size \(n\) from a population with mean \(\mu\) and known variance \(\sigma^{2}\), then a \((1-\alpha) 100 \%\) confidence interval for \(\mu\) is \begin{eqnarray*} \left(\bar{X} - z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}, \bar{X} + z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}\right) &\Longleftrightarrow &\bar{X} \pm z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \\ &\Longleftrightarrow &\bar{X}-z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \lt \mu \lt \bar{X}+z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \end{eqnarray*}
    2. where \(z_{\frac{\alpha}{2}}\) is the \(\mathrm{z}\) -value leaving an area of \(\alpha / 2\) to the right; i.e., \(\mathrm{P}\big(\mathrm{Z}\gt z_{\frac{\alpha}{2}}\big)=\alpha / 2,\) or equivalently, \(\mathrm{P}\big(\mathrm{Z}\lt z_{\frac{\alpha}{2}}\big)=1-\alpha / 2\)

ឧទាហរណ៍.

មធ្យមរបស់កំហាប់ស័ង្កសី ត្រូវបានកត់ត្រាយកចេញពីសំណាកមួយនៃការវាស់កម្រិតស័ង្កសី នៅ $36$ ទីតាំងផ្សេងគ្នា ដែលមានកម្រិត $2.6$ ក្រាម/មីលីលីត្រ។ ចូររកចន្លោះពេលជឿជាក់ (C.I. ) \(95 \%\) និង \(99 \%\) សម្រាប់មធ្យមនៃកំហាប់ស័ង្កសី នៅក្នុងទឹកទន្លេ។ សន្មតថា គម្លាតស្តង់ដារបស់ប៉ូពុយឡាស្យុង \(0.3\) ។

Theorem. If \(\bar{X}\) is used as an estimate of \(\mu,\) we can then be \((1-\alpha) 100 \%\) confident that the error (in estimation) will not exceed max error of estimation $E=z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}$.

Remark.

  • If the maximum amount of the error \(E=z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}\), then we can find the sample size $n$ by $$ E=z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \Longleftrightarrow \sqrt{n}=z_{\frac{\alpha}{2}} \frac{\sigma}{E} \Longleftrightarrow n=\left(z_{\frac{\alpha}{2}} \frac{\sigma}{E}\right)^{2} $$
  • If \(\sigma\) is unknown, we could take a preliminary sample of size \(n \geq 30\) to provide an estimate of \(\sigma\). Then using \(S=\sqrt{\sum_{i=1}^{n}\left(X_{i}-\bar{X}\right)^{2} /(n-1)}\) as an approximation for \(\sigma\), we could determine approximately how many observations are needed to provide the desired degree of accuracy.

  1. Second Case: $\sigma^2$ is unknown.
    2. If \(\bar{X}=\sum_{i=1}^{n} X_{i} / n\) and \(S=\sqrt{\sum_{i=1}^{n}\left(X_{i}-\bar{X}\right)^{2} /(n-1)}\) are the sample mean and the sample standard deviation of a random sample of size \(n\) from a normal population (distribution) with unknown variance \(\sigma^{2}\), then \(a(1-\alpha) 100 \%\) confidence interval for \(\mu\) is : $$ \left(\bar{X}-t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}, \bar{X}+t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}\right) \Longleftrightarrow \bar{X} \pm t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} \Longleftrightarrow \bar{X} - t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} \lt \mu \lt \bar{X}+t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} $$ where \(t_{\frac{\alpha}{2}}\) is the $t$-value with \(df=n-1\) leaving an area of \(\alpha / 2\) to the right; i.e., \(\mathrm{P}\big(\mathrm{T} \gt t_{\frac{\alpha}{2}}\big)=\alpha / 2,\) or equivalently, \(\mathrm{P}\big(\mathrm{T} \lt t_{\frac{\alpha}{2}}\big)=1-\alpha / 2\).

The contents of $7$ similar containers of sulfuric acid are $9.8, 10.2, 10.4, 9.8, 10.0, 10.2$ and $9.6$ liters. Find a $95\%$ C.I. for the mean of all such containers, assuming an approximate normal distribution.

ឧទាហរណ៍.

Consider the following data: $$64.3, 64.6, 64.8, 64.2, 64.5, 64.3, 64.6, 64.8, 64.2, 64.3$$ Assume that it is normally distributed with unit population variance. For these data, construct a $95\%$ confidence interval for the population mean.

ចម្លើយ. Using the data, $n=10, \bar{x}=64.46,$ the level of significance, $\alpha=5 \%=0.05,$ and from the given assumption, $\sigma^{2}=1 .$ Now, the resulting $95 \%$ confidence interval (CI) for the population mean is \[ \bar{x}-z_{0.025} \times \frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{x}+z_{0.025} \times \frac{\sigma}{\sqrt{n}} \]

Standard Error of a Point Estimate

  • The standard error of an estimator is its standard deviation.
  • We use \(\bar{X}=\sum_{i=1}^{n} X_{i} / n\) as a point estimator of \(\mu,\) and we used the sampling distribution of \(\bar{X}\) to make a \((1-\alpha) 100 \%\) C.I. for \(\mu\).
  • The standard deviation of \(\bar{X},\) which is \(\sigma_{\bar{X}}=\sigma / \sqrt{n},\) is called the standard error of \(\bar{X} .\) We write \(s.e (\bar{X})=\sigma / \sqrt{n}\)
  • \((1-\alpha) 100 \%\) C.I. for \(\mu,\) when \(\sigma^{2}\) is known, is $$ \bar{X} \pm Z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}=\bar{X} \pm Z_{\frac{\alpha}{2}}\times s .e (\bar{X}) $$
  • \((1-\alpha) 100 \%\) C.I. for \(\mu,\) when \(\sigma^{2}\) is unknown, and the distribution is normal, is $$ \bar{X} \pm t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}=\bar{X} \pm t_{\frac{\alpha}{2}}\times s .e (\bar{X}) \quad \text{ with } df = n-1 $$

3. Two Samples: Estimate the Difference between Two Means ($\mu_1 - \mu_2$)

Recall: For two independent samples: \begin{eqnarray*} \mu_{\bar{X}_{1}-\bar{X}_{2}} &=&\mu_{1}-\mu_{2}=E\left(\bar{X}_{1}-\bar{X}_{2}\right); \qquad \sigma_{\bar{X}_{1}-\bar{X}_{2}}^{2} = \frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}=\operatorname{Var}\left(\bar{X}_{1}-\bar{X}_{2}\right)\\ \sigma_{\bar{X}_{1}-\bar{X}_{2}} &=& \sqrt{\sigma_{\bar{X}_{1}-\bar{X}_{2}}^{2}}=\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}};\qquad Z = \frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim N(0,1) \end{eqnarray*}

  • Point Estimation of $\mu_1 - \mu_2$. The difference $\overline{X_1} - \overline{X_2}$ is a "good" point estimate for \(\mu_1 - \mu_2\).
  • Interval Estimation (Confidence Interval) of $\mu_1 - \mu_2$.
  1. First Case: $\sigma_1^2$ and $\sigma_2^2$ are known.
    • Let \(Z = \frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim N(0,1)\)
    • Then a $(1-\alpha)100\%$ confidence interval for $\mu_1 - \mu_2$ is $$ \left(\bar{X}_{1}-\bar{X}_{2}\right)-Z_{\frac{\alpha}{2}} \sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}} \lt \mu_{1}-\mu_{2} \lt \left(\bar{X}_{1} - \bar{X}_{2}\right)+Z_{\frac{\alpha}{2}} \sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}} $$
  1. Second Case: $\sigma_1^2=\sigma_2^2=\sigma^2$ is unknown.
    • If \(\sigma_{1}^{2}\) and \(\sigma_{2}^{2}\) are unknown but \(\sigma_{1}^{2}=\sigma_{2}^{2}=\sigma^{2},\) then the pooled estimate of \(\sigma^{2}\) is $$ S_{p}^{2}=\frac{\left(n_{1}-1\right) S_{1}^{2}+\left(n_{2}-1\right) S_{2}^{2}}{n_{1}+n_{2}-2} $$ where \(S_{1}^{2}\) is the variance of the 1-st sample and \(S_{2}^{2}\) is the variance of the 2-nd sample. The degrees of freedom of \(S_{p}^{2}\) is \(df=n_{1}+n_{2}-2\).
    • Then a $(1-\alpha)100\%$ confidence interval for $\mu_1 - \mu_2$ is $$ \left(\bar{X}_{1}-\bar{X}_{2}\right)-t_{\frac{\alpha}{2}} \sqrt{\frac{S_{p}^{2}}{n_{1}}+\frac{S_{p}^{2}}{n_{2}}} \lt \mu_{1}-\mu_{2} \lt \left(\bar{X}_{1}-\bar{X}_{2}\right)+t_{\frac{\alpha}{2}} \sqrt{\frac{S_{p}^{2}}{n_{1}}+\frac{S_{p}^{2}}{n_{2}}} $$ where \(t_{\frac{\alpha}{2}}\) is the $t$-value with \(df=n_{1}+n_{2}-2\) degrees of freedom.

Single Sample: Estimating of a Proportion

  • $p =$ Population proportion of successes (elements of Type A) in the population. $$=\frac{A}{A+B} = \frac{\text { no. of elements of type } A \text { in the population }}{\text { Total no. of elements }} $$
  • $n =$ sample size
  • $X =$ no. of elements of type $A$ in the sample of size $n$.
  • $\hat{p} =$ Sample proportion of successes (elements of Type A) in the sample $=\frac{X}{n}$

រំលឹក :

  • $X \sim \operatorname{Binomial}(n, p)$
  • $\mathrm{E}(\hat{p})=\mathrm{E}\left(\frac{X}{n}\right)=p$
  • $\operatorname{Var}(\hat{p})=\operatorname{Var}\left(\frac{X}{n}\right)=\frac{p q}{n}; \quad q=1-p$
  • For large $n$, we have $$ \begin{aligned} \hat{p} &\sim \mathrm{N}\left(p, \sqrt{\frac{p q}{n}}\right) &\text { (Approximately) } \\ Z&=\frac{\hat{p}-p}{\sqrt{\frac{p q}{n}}} \sim \mathrm{N}(0,1) &\text { (Approximately) } \end{aligned}$$

  • Point Estimation for $p$: A good point estimator for the population proportion $p$ is given by the statistic (sample proportion): $\hat{p} = \frac{X}{n}$
  • Confidence Interval for $p$: For large $n$, an approximate $(1-\alpha)100\%$ confidence interval for $p$ is : $$ \hat{p}-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}} \lt p \lt \hat{p}+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}}; \quad \hat{q} = 1 - \hat{p} $$

ឧទាហរណ៍.

In a random sample of $n=500$ families owing television sets in the city of Hamilton, Canada, it was found that $x=340$ subscribed to HBO. Find $95\%$ confidence interval for the actual proportion of families in this city who subscribe to HBO.

Two Samples: Estimating the Difference between Two Proportions:

Suppose that we have two populations:
  • $p_1 =$ proportion of the 1-st population.
  • $p_2 =$ proportion of the 2-nd population.
  • We are interested in comparing $p_1$ and $p_2$, or equivalently, making inferences about $p_1- p_2$.
  • We independently select a random sample of size $n_1$ from the 1-st population and another random sample of size $n_2$ from the 2-nd population:
  • Let $X_1 =$ no. of elements of type A in the 1-st sample: $X_1\sim \operatorname{Binomial}(n_1, p_1)$
  • Let $X_2 =$ no. of elements of type A in the 2-nd sample: $X_2\sim \operatorname{Binomial}(n_2, p_2)$
  • $\hat{p}_{1}=\frac{X_{1}}{n_{1}}=$ proportion of the 1-st sample
  • $\hat{p}_{2}=\frac{X_{2}}{n_{2}}=$ proportion of the 2-st sample

តាមសម្មតិកម្មខាងលើ គេបានកំណត់បានលទ្ធផល ដូចតទៅ៖

  1. $E\left(\hat{p}_{1}-\hat{p}_{2}\right)=p_{1}-p_{2}$
  2. $\operatorname{Var}\left(\hat{p}_{1}-\hat{p}_{2}\right)=\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}} ;\quad q_{1}=1-p_{1}, q_{2}=1-p_{2}$
  3. For large $n_{1}$ and $n_{2}$, we have \begin{eqnarray*} \hat{p}_{1}-\hat{p}_{2} &\sim & N\left(p_{1}-p_{2}, \sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}\right); \quad \text{ (Approximately) } \\ Z &=&\frac{\left(\hat{p}_{1}-\hat{p}_{2}\right)-\left(p_{1}-p_{2}\right)}{\sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}} \sim \mathrm{N}(0,1); \quad \text { (Approximately) } \end{eqnarray*}

  • Point Estimation of $p_1 - p_2$. A good point estimator for the difference between the two proportions, $p_1- p_2$, is given by the statistic: $$ \hat{p}_{1}-\hat{p}_{2}=\frac{X_{1}}{n_{1}}-\frac{X_{2}}{n_{2}} $$
  • Confidence Interval for $p_1 - p_2$. For large \(n_{1}\) and \(n_{2},\) an approximate \((1-\alpha) 100 \%\) confidence interval for \(p_{1}-p_{2}\) is : $$ \left(\hat{p}_{1}-\hat{p}_{2}\right)-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} \lt p_{1}-p_{2} \lt \left(\hat{p}_{1}-\hat{p}_{2}\right)+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} $$

ឧទាហរណ៍.

A certain change in a process for manufacture of component parts is being considered. Samples are taken using both existing and the new procedure to determine if the new process results in an improvement. If $75$ of $1500$ items from the existing procedure were found to be defective and $80$ of $2000$ items from the new procedure were found to be defective, find $90\%$ confidence interval for the true difference in the fraction of defectives between the existing and the new process.

Single Sample: Estimating of a Proportion

  • $p =$ Population proportion of successes (elements of Type A) in the population. $$=\frac{A}{A+B} = \frac{\text { no. of elements of type } A \text { in the population }}{\text { Total no. of elements }} $$
  • $n =$ sample size
  • $X =$ no. of elements of type $A$ in the sample of size $n$.
  • $\hat{p} =$ Sample proportion of successes (elements of Type A) in the sample $=\frac{X}{n}$

រំលឹក :

  • $X \sim \operatorname{Binomial}(n, p)$
  • $\mathrm{E}(\hat{p})=\mathrm{E}\left(\frac{X}{n}\right)=p$
  • $\operatorname{Var}(\hat{p})=\operatorname{Var}\left(\frac{X}{n}\right)=\frac{p q}{n}; \quad q=1-p$
  • For large $n$, we have $$ \begin{aligned} \hat{p} &\sim \mathrm{N}\left(p, \sqrt{\frac{p q}{n}}\right) &\text { (Approximately) } \\ Z&=\frac{\hat{p}-p}{\sqrt{\frac{p q}{n}}} \sim \mathrm{N}(0,1) &\text { (Approximately) } \end{aligned}$$

  • Point Estimation for $p$: A good point estimator for the population proportion $p$ is given by the statistic (sample proportion): $\hat{p} = \frac{X}{n}$
  • Confidence Interval for $p$: For large $n$, an approximate $(1-\alpha)100\%$ confidence interval for $p$ is : $$ \hat{p}-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}} \lt p \lt \hat{p}+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}}; \quad \hat{q} = 1 - \hat{p} $$
