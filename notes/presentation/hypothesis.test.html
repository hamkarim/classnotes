  3. Identify Type I and Type II errors and interpret them

សេចក្ដីផ្ដើម

Consider a population with some unknown parameter $\theta$. We are interested in testing (confirming or denying) some conjectures about $\theta$. For example, we might be interested in testing the conjecture that $\theta = \theta_0$ where $\theta_0$ is a given value.

  • A statistical hypothesis is a conjecture concerning (or a statement about) the population.
  • For example, if \(\theta\) is an unknown parameter of the population, we may be interested in testing the conjecture that \(\theta>\theta_{\mathrm{o}}\) for some specific value \(\theta_{\mathrm{o}}\).

Goal: We usually test the null hypothesis: $$\mathrm{H}_{\mathrm{o}}: \theta=\theta_{\mathrm{o}} \quad \text { (Null Hypothesis) }$$ We have one of the following alternative hypotheses: $$\mathrm{H}_{1}: \begin{array}{ll} \left\{ \begin{array}{l} \theta \neq \theta_{\mathrm{o}} \\ \theta \gt \theta_{\mathrm{o}} \\ \theta \lt \theta_{\mathrm{o}} \end{array}\right. & \begin{array}{l} \text { (Alternative Hypothesis }\\ \text { or Research Hypothesis }) \end{array} \end{array}$$

Possible situations in testing a statistical hypothesis
$$ \begin{array}{|l|c|c|} \hline & \mathrm{H}_{0} \text { is true } & \mathrm{H}_{0} \text { is false } \\ \hline \text { Accepting } \mathrm{H}_{0} & \text { Correct Decision } & \begin{array}{c} \text { Type II error } \\ (\beta)\end{array} \\ \hline \text { Rejecting } \mathrm{H}_{0} & \begin{array}{c} \text { Type I error } \\ (\alpha)\end{array} & \text { Correct Decision } \\ \hline\end{array} $$

១.  ចន្លោះជឿជាក់ (Confidence Intervals)

ជាការកត់សម្គាល់ យើងមិនអាចប្រាកដថាចន្លោះមួយ មានតម្លៃប៉ារ៉ាម៉ែត្ររបស់ប៉ូពុយឡាស្យុង ពិតប្រាកដ ប៉ុន្តែមិនស្គាល់នោះទេ ដោយសារតែ យើងប្រើតែសំណាកមួយ ដែលយកចេញពីរបាយប៉ូពុយឡាស្យុងទាំងមូល ដើម្បីយកមកគណនារកតម្លៃប៉ាន់ស្មានជាចំណុច និង តម្លៃប៉ាន់ស្មានជាចន្លោះ។ ក្នុងករណីបែបនេះ គេត្រូវសង់ ចន្លោះជឿជាក់មួយ ដើម្បីអោយមានភាពប្រាកដច្បាស់ថា ចន្លោះនេះមានផ្ទុកនូវប៉ារ៉ាម៉ែត្ររបស់ប៉ូពុយឡាស្យុងពិតមួយ ប៉ុន្តែមិនស្គាល់តម្លៃ។ ជាទូទៅ ចន្លោះជឿជាក់ (CI) ចំនួន $100(1 - \alpha)\%$ ស្មើនឹង

$ \text{point estimate }\pm \text{ reliability coefficient } \times \text{ s.e.(parameter)} $
  • $\alpha$ ជា level of significance មានតម្លៃចន្លោះ $0$ និង $1$ ហើយតម្លៃ $1-\alpha$ ហៅថា "confidence coefficient";
  • $100(1 - \alpha)\%$ ហៅថា confidence level;
  • parameter estimate គឺជាតម្លៃមួយសម្រាប់ចំណុចប៉ាន់ស្មាន ដូចជា $\overline{x}$ ប្រើសម្រាប់មធ្យមសំណាក ឬ $\widehat{p}$ សម្រាប់សមាមាត្ររបស់ប៉ូពុយឡាស្យុង។
  • reliability coefficient គឺជាតម្លៃប្រូបាប៊ីលីតេមួយ ដែលទទួលបានពីតារាងមួយត្រឹមត្រូវ ដូចជា តារាង $z_{\alpha}$ ឬតារាង $t_{(\frac{\alpha}{2},n-1)}$។
  • s.e.(parameter) អានថា standard error of the parameter គឺជាតម្លៃមួយវាស់បាននូវគម្លាតតូច-ធំរបស់ចំណុចប៉ាន់ស្មានទៅតម្លៃប៉ារ៉ាម៉ែត្រប៉ូពុយឡាស្យុងពិតប្រាកដ។


Classical Methods of Estimation ( និយមន័យ)

  1. ចំណុចប៉ាន់ស្មាន (point Estimation). ចំណុចប៉ាន់ស្មានរបស់ប៉ារ៉ាម៉ែត្រប៉ូពុយឡាស្យុង \(\theta\) គឺជាតម្លៃ \(\hat{\theta}\) របស់តម្លៃស្ថិតិមួយ \(\hat{\Theta}\)។ ដូចជា តម្លៃ \(\overline{x}\) របស់ស្ថិតិ \(\overline{X}\) ដែលគណនាបានពីសំណាកមួយទំហំ $n$ គឺជាចំណុចប៉ាន់ស្មានរបស់មធ្យមប៉ូពុយឡាស្យុង \(\mu\)។
  2. ចន្លោះប៉ាន់ស្មាន (Interval Estimation) ឬ (Confidence Interval = C.I). ចន្លោះប៉ាន់ស្មាន​របស់ប៉ារ៉ាម៉ែត្រ​ប៉ូពុយឡាស្យុង \(\theta\) មានរាង \(\big(\hat{\theta}_{L}, \hat{\theta}_{U}\big)\) មានន័យថា \(\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}\)។ ចន្លោះនេះផ្ទុកតម្លៃពិតប្រាកដរបស់ \(\theta\) មានប្រូបាប៊ីលីតេ \(\mathrm{P}\left(\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}\right)=1-\alpha\)។

  • $\hat{\theta}_{L}\lt\theta\lt\hat{\theta}_{U}$ is called a $(1-\alpha)100\%$ confidence interval (C.I.) for $\theta$.
  • $\theta_{L}$ is lower confidence limit, and $\theta_{U}$ is upper confidence limit
  • $1-\alpha$ is called the confidence coefficient; $\alpha= 1-\text{ level of confidence}$, $(0\lt \alpha\lt 1)$

For example, you might have seen a problem like this: Let $X$ be a normal random variable with mean $\mu =100$ and variance $\sigma^2=15$. Find the probability that $X \gt 110$.

In real life, we might not know the distribution of $X$, so we need to collect data, and from the data we should conclude whether $X$ has a normal distribution or not.

Critical Value

  • $z_{\frac{\alpha}{2}} =$ critical value for the standard normal distribution ($z$-distribution)
  • $t_{(\frac{\alpha}{2}, n-1)} =$ critical value for the $t$-distribution with $df = n-1$

2. Single Sample - Estimation of the Mean ($\mu$)

យើងសន្មតជាទូទៅថា ទិន្នន័យជាសំណាក គឺបានមកពីប៉ូពុយឡាយស្យុងធម្មតាមួយ។

Recall: $$ \begin{array}{l} E(\overline{X})=\mu_{\overline{X}}=\mu; \quad \operatorname{Var}\big(\overline{X}\big)=\sigma_{\overline{X}}^{2}=\frac{\sigma^{2}}{n}; \quad \overline{X} \sim N\left(\mu, \frac{\sigma}{\sqrt{n}}\right) \\ Z=\frac{\overline{X}-\mu}{\sigma / \sqrt{n}} \sim \mathrm{N}(0,1) \text{ for } \left(\sigma^{2} \text { is known }\right); \quad T=\frac{\overline{X}-\mu}{S / \sqrt{n}} \sim \mathrm{t}(n-1) \text{ for } \left(\sigma^{2} \text { is unknown }\right) \end{array} $$ ក្នុងករណីនេះ យើងកំណត់តម្លៃសំណាករបាយអថេរចៃដន្យ $\overline{X}$ ដើម្បីធ្វើសេចក្ដីសន្និដ្ឋានរកតម្លៃប៉ារ៉ាម៉ែត្រ $\mu$ ។

  • Point Estimation of $\mu$. The sample mean \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) is a "good" point estimate for \(\mu\).
  • Interval Estimation (Confidence Interval) of $\mu$.
  1. First Case: $\sigma^2$ is known.

    If \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) is the sample mean of a random sample of size \(n\) from a population with mean \(\mu\) and known variance \(\sigma^{2}\), then a \((1-\alpha) 100 \%\) confidence interval for \(\mu\) is \begin{eqnarray*} \left(\overline{X} - z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}, \overline{X} + z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}\right) &\Longleftrightarrow &\overline{X} \pm z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \\ &\Longleftrightarrow &\overline{X}-z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \lt \mu \lt \overline{X}+z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}} \end{eqnarray*}

    2. where \(z_{\frac{\alpha}{2}}\) is the \(\mathrm{z}\) -value leaving an area of \(\alpha / 2\) to the right; i.e., \(\mathrm{P}\big(\mathrm{Z}\gt z_{\frac{\alpha}{2}}\big)=\alpha / 2,\) or equivalently, \(\mathrm{P}\big(\mathrm{Z}\lt z_{\frac{\alpha}{2}}\big)=1-\alpha / 2\)

ឧទាហរណ៍.

មធ្យមរបស់កំហាប់ស័ង្កសី ត្រូវបានកត់ត្រាយកចេញពីសំណាកមួយនៃការវាស់កម្រិតស័ង្កសី នៅ $36$ ទីតាំងផ្សេងគ្នា ដែលមានកម្រិត $2.6$ ក្រាម/មីលីលីត្រ។ ចូររកចន្លោះពេលជឿជាក់ (C.I. ) \(95 \%\) និង \(99 \%\) សម្រាប់មធ្យមនៃកំហាប់ស័ង្កសី នៅក្នុងទឹកទន្លេ។ សន្មតថា គម្លាតស្តង់ដារបស់ប៉ូពុយឡាស្យុង \(0.3\) ។

Determining the Sample Size

Theorem. If \(\overline{X}\) is used as an estimate of \(\mu,\) we can then be \((1-\alpha) 100 \%\) confident that the error (in estimation) will not exceed max error of estimation $z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}$, called Margin of Error (MOE) or Sampling Error (SE).

Remark.

  • If the MOE, \(E=z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}\), for $\mu$, then we can find the sample size $n$ by $n=\big(z_{\frac{\alpha}{2}} \frac{\sigma}{E}\big)^{2}$
  • Sample size for $100(1 – \alpha)\%$ Confidence Interval for $p$ with MOE, $E=z_{\frac{\alpha}{2}} \sqrt{\frac{pq}{n}}$, is $n=\frac{z_{\alpha / 2}^{2} \widehat{p} \widehat{q}}{E^{2}}$
  • If \(\sigma\) is unknown, we could take a preliminary sample of size \(n \geq 30\) to provide an estimate of \(\sigma\). Then using \(S=\sqrt{\sum_{i=1}^{n}\left(X_{i}-\overline{X}\right)^{2} /(n-1)}\) as an approximation for \(\sigma\), we could determine approximately how many observations are needed to provide the desired degree of accuracy.

  1. Second Case: $\sigma^2$ is unknown.

    2. If \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) and \(S=\sqrt{\sum_{i=1}^{n}\left(X_{i}-\overline{X}\right)^{2} /(n-1)}\) are the sample mean and the sample standard deviation of a random sample of size \(n\) from a normal population (distribution) with unknown variance \(\sigma^{2}\), then \(a(1-\alpha) 100 \%\) confidence interval for \(\mu\) is : $$ \left(\overline{X}-t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}, \overline{X}+t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}\right) \Longleftrightarrow \overline{X} \pm t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} \Longleftrightarrow \overline{X} - t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} \lt \mu \lt \overline{X}+t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}} $$ where \(t_{\frac{\alpha}{2}}\) is the $t$-value with \(df=n-1\) leaving an area of \(\alpha / 2\) to the right; i.e., \(\mathrm{P}\big(\mathrm{T} \gt t_{\frac{\alpha}{2}}\big)=\alpha / 2,\) or equivalently, \(\mathrm{P}\big(\mathrm{T} \lt t_{\frac{\alpha}{2}}\big)=1-\alpha / 2\).

Standard Error of a Point Estimate

  • The standard error of an estimator is its standard deviation.
  • We use \(\overline{X}=\sum_{i=1}^{n} X_{i} / n\) as a point estimator of \(\mu,\) and we used the sampling distribution of \(\overline{X}\) to make a \((1-\alpha) 100 \%\) C.I. for \(\mu\).
  • The standard deviation of \(\overline{X},\) which is \(\sigma_{\overline{X}}=\sigma / \sqrt{n},\) is called the standard error of \(\overline{X} .\) We write \(s.e (\overline{X})=\sigma / \sqrt{n}\)
  • \((1-\alpha) 100 \%\) C.I. for \(\mu,\) when \(\sigma^{2}\) is known, is $$ \overline{X} \pm Z_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{n}}=\overline{X} \pm Z_{\frac{\alpha}{2}}\times s .e (\overline{X}) $$
  • \((1-\alpha) 100 \%\) C.I. for \(\mu,\) when \(\sigma^{2}\) is unknown, and the distribution is normal, is $$ \overline{X} \pm t_{\frac{\alpha}{2}} \frac{S}{\sqrt{n}}=\overline{X} \pm t_{\frac{\alpha}{2}}\times s .e (\overline{X}) \quad \text{ with } df = n-1 $$

3. Two Samples: Estimate the Difference between Two Means ($\mu_1 - \mu_2$)

Recall: For two independent samples: \begin{eqnarray*} \mu_{\overline{X}_{1}-\overline{X}_{2}} &=&\mu_{1}-\mu_{2}=E\left(\overline{X}_{1}-\overline{X}_{2}\right); \qquad \sigma_{\overline{X}_{1}-\overline{X}_{2}}^{2} = \frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}=\operatorname{Var}\left(\overline{X}_{1}-\overline{X}_{2}\right)\\ \sigma_{\overline{X}_{1}-\overline{X}_{2}} &=& \sqrt{\sigma_{\overline{X}_{1}-\overline{X}_{2}}^{2}}=\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}};\qquad Z = \frac{\left(\overline{X}_{1}-\overline{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim N(0,1) \end{eqnarray*}

  • Point Estimation of $\mu_1 - \mu_2$. The difference $\overline{X_1} - \overline{X_2}$ is a "good" point estimate for \(\mu_1 - \mu_2\).
  • Interval Estimation (Confidence Interval) of $\mu_1 - \mu_2$.
  1. ករណីទី១ : ស្គាល់តម្លៃ $\sigma_1^2$ និង $\sigma_2^2$
    • Let \(Z = \frac{\left(\overline{X}_{1}-\overline{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim N(0,1)\)
    • Then a $(1-\alpha)100\%$ confidence interval for $\mu_1 - \mu_2$ is $$ \left(\overline{X}_{1}-\overline{X}_{2}\right)-Z_{\frac{\alpha}{2}} \sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}} \lt \mu_{1}-\mu_{2} \lt \left(\overline{X}_{1} - \overline{X}_{2}\right)+Z_{\frac{\alpha}{2}} \sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}} $$
  1. ករណីទី២ : $\sigma_1^2=\sigma_2^2=\sigma^2$ មិនស្គាល់តម្លៃ
    • If \(\sigma_{1}^{2}\) and \(\sigma_{2}^{2}\) are unknown but \(\sigma_{1}^{2}=\sigma_{2}^{2}=\sigma^{2},\) then the pooled estimate of \(\sigma^{2}\) is $$ S_{p}^{2}=\frac{\left(n_{1}-1\right) S_{1}^{2}+\left(n_{2}-1\right) S_{2}^{2}}{n_{1}+n_{2}-2} $$ where \(S_{1}^{2}\) is the variance of the 1-st sample and \(S_{2}^{2}\) is the variance of the 2-nd sample. The degrees of freedom of \(S_{p}^{2}\) is \(df=n_{1}+n_{2}-2\).
    • Then a $(1-\alpha)100\%$ confidence interval for $\mu_1 - \mu_2$ is $$ \left(\overline{X}_{1}-\overline{X}_{2}\right)-t_{\frac{\alpha}{2}} \sqrt{\frac{S_{p}^{2}}{n_{1}}+\frac{S_{p}^{2}}{n_{2}}} \lt \mu_{1}-\mu_{2} \lt \left(\overline{X}_{1}-\overline{X}_{2}\right)+t_{\frac{\alpha}{2}} \sqrt{\frac{S_{p}^{2}}{n_{1}}+\frac{S_{p}^{2}}{n_{2}}} $$ where \(t_{\frac{\alpha}{2}}\) is the $t$-value with \(df=n_{1}+n_{2}-2\) degrees of freedom.

4. Single Sample: Estimating of a Proportion

  • $p =$ Population proportion of successes (elements of Type A) in the population. $$=\frac{A}{A+B} = \frac{\text { no. of elements of type } A \text { in the population }}{\text { Total no. of elements }} $$
  • $n =$ sample size
  • $X =$ no. of elements of type $A$ in the sample of size $n$.
  • $\hat{p} =$ Sample proportion of successes (elements of Type A) in the sample $=\frac{X}{n}$

រំលឹក :

  • $X \sim \operatorname{Binomial}(n, p)$
  • $\mathrm{E}(\hat{p})=\mathrm{E}\left(\frac{X}{n}\right)=p$
  • $\operatorname{Var}(\hat{p})=\operatorname{Var}\left(\frac{X}{n}\right)=\frac{p q}{n}; \quad q=1-p$
  • For large $n$, we have $$ \begin{aligned} \hat{p} &\sim \mathrm{N}\left(p, \sqrt{\frac{p q}{n}}\right) &\text { (Approximately) } \\ Z&=\frac{\hat{p}-p}{\sqrt{\frac{p q}{n}}} \sim \mathrm{N}(0,1) &\text { (Approximately) } \end{aligned}$$

  • Point Estimation for $p$: A good point estimator for the population proportion $p$ is given by the statistic (sample proportion): $\hat{p} = \frac{X}{n}$
  • Confidence Interval for $p$: For large $n$, an approximate $(1-\alpha)100\%$ confidence interval for $p$ is : $$ \hat{p}-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}} \lt p \lt \hat{p}+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}}; \quad \hat{q} = 1 - \hat{p} $$

5. Two Samples: Estimating the Difference between Two Proportions

Suppose that we have two populations:
  • $p_1 =$ proportion of the 1-st population.
  • $p_2 =$ proportion of the 2-nd population.
  • We are interested in comparing $p_1$ and $p_2$, or equivalently, making inferences about $p_1- p_2$.
  • We independently select a random sample of size $n_1$ from the 1-st population and another random sample of size $n_2$ from the 2-nd population:
  • Let $X_1 =$ no. of elements of type A in the 1-st sample: $X_1\sim \operatorname{Binomial}(n_1, p_1)$
  • Let $X_2 =$ no. of elements of type A in the 2-nd sample: $X_2\sim \operatorname{Binomial}(n_2, p_2)$
  • $\hat{p}_{1}=\frac{X_{1}}{n_{1}}=$ proportion of the 1-st sample
  • $\hat{p}_{2}=\frac{X_{2}}{n_{2}}=$ proportion of the 2-st sample

តាមសម្មតិកម្មខាងលើ គេបានកំណត់បានលទ្ធផល ដូចតទៅ៖

  1. $E\left(\hat{p}_{1}-\hat{p}_{2}\right)=p_{1}-p_{2}$
  2. $\operatorname{Var}\left(\hat{p}_{1}-\hat{p}_{2}\right)=\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}} ;\quad q_{1}=1-p_{1}, q_{2}=1-p_{2}$
  3. For large $n_{1}$ and $n_{2}$, we have \begin{eqnarray*} \hat{p}_{1}-\hat{p}_{2} &\sim & N\left(p_{1}-p_{2}, \sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}\right); \quad \text{ (Approximately) } \\ Z &=&\frac{\left(\hat{p}_{1}-\hat{p}_{2}\right)-\left(p_{1}-p_{2}\right)}{\sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}} \sim \mathrm{N}(0,1); \quad \text { (Approximately) } \end{eqnarray*}

  • Point Estimation of $p_1 - p_2$. A good point estimator for the difference between the two proportions, $p_1- p_2$, is given by the statistic: $$ \hat{p}_{1}-\hat{p}_{2}=\frac{X_{1}}{n_{1}}-\frac{X_{2}}{n_{2}} $$
  • Confidence Interval for $p_1 - p_2$. For large \(n_{1}\) and \(n_{2},\) an approximate \((1-\alpha) 100 \%\) confidence interval for \(p_{1}-p_{2}\) is : $$ \left(\hat{p}_{1}-\hat{p}_{2}\right)-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} \lt p_{1}-p_{2} \lt \left(\hat{p}_{1}-\hat{p}_{2}\right)+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} $$
