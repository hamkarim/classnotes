សេចក្ដីផ្ដើម

Consider a population with some unknown parameter $\theta$. We are interested in testing (confirming or denying) some conjectures about $\theta$. For example, we might be interested in testing the conjecture that $\theta = \theta_0$ where $\theta_0$ is a given value.

  • A statistical hypothesis is a conjecture concerning (or a statement about) the population.

    • For example, if \(\theta\) is an unknown parameter of the population, we may be interested in testing the conjecture that \(\theta>\theta_{0}\) for some specific value \(\theta_{0}\).


  • Null hypothesis, $H_0$, is a statistical hypothesis that states that there is no difference between a parameter and a specific value, or that there is no difference between two parameters.

  • Alternative hypothesis, $H_1$, is a statistical hypothesis that states the existence of a difference between a parameter and a specific value, or states that there is a difference between two parameters

Hypothesis-Testing Common Phrases

$$ \begin{array}{ll} \qquad \gt & \qquad \lt \\ \text { Is greater than } & \text { Is less than } \\ \text { Is above } & \text { Is below } \\ \text { Is higher than } & \text { Is lower than } \\ \text { Is longer than } & \text { Is shorter than } \\ \text { Is bigger than } & \text { Is smaller than } \\ \text { Is increased } & \text { Is decreased or reduced from } \\ \qquad \geq & \qquad\leq \\ \text { Is greater than or equal to } & \text { Is less than or equal to } \\ \text { Is at least } & \text { Is at most } \\ \text { Is not less than } & \text { Is not more than } \\ \qquad = & \qquad \neq \\ \text { Is equal to } & \text { Is not equal to } \\ \text { Is exactly the same as } & \text { Is different from } \\ \text { Has not changed from } & \text { Has changed from } \\ \text { Is the same as } & \text { Is not the same as } \end{array} $$

Examples:

  • A medical researcher is interested in finding out whethera new medication will have any undesirable side effects. The researcher is particularly concerned with the pulserate of the patients who take the medication. Will the pulse rate increase, decrease, or remain unchanged after a patient takes the medication? The researcher knows that the mean pulse rate for the population under study is $82$ beats per minute.

  • A chemist invents an additive to increase the life of an automobile battery. The mean lifetime of the automobile battery without the additive is 36 months.

  • A contractor wishes to lower heating bills by using aspecial type of insulation in houses. If the average of the monthly heating bills is $\$78$, what is the hypotheses about heating costs with the use of insulation?

Goal of Hypothesis Testing

Our goal: is to test the null hypothesis: $$\mathrm{H}_{\mathrm{o}}: \theta=\theta_{\mathrm{o}} \quad \text { (Null Hypothesis) }$$ against one of the following alternative hypotheses: $$\mathrm{H}_{1}: \begin{array}{ll} \left\{ \begin{array}{l} \theta \neq \theta_{\mathrm{o}} \\ \theta \gt \theta_{\mathrm{o}} \\ \theta \lt \theta_{\mathrm{o}} \end{array}\right. & \begin{array}{l} \text { (Alternative Hypothesis }\\ \text { or Research Hypothesis }) \end{array} \end{array}$$

Possible situations in testing a statistical hypothesis
$$ \begin{array}{|l|c|c|} \hline & \mathrm{H}_{0} \text { is true } & \mathrm{H}_{0} \text { is false } \\ \hline \text { Accepting } \mathrm{H}_{0} & \text { Correct Decision } & \begin{array}{c} \text { Type II error } \\ (\beta)\end{array} \\ \hline \text { Rejecting } \mathrm{H}_{0} & \begin{array}{c} \text { Type I error } \\ (\alpha)\end{array} & \text { Correct Decision } \\ \hline\end{array} $$ The table tells us that,
  • Type I error $=$ Rejecting $\mathrm{H}_{0}$ when $\mathrm{H}_{0}$ is true
  • Type II error $=$ Accepting $\mathrm{H}_{0}$ when $\mathrm{H}_{0}$ is false
  • $\text{ P(Type I error) }$$=\mathrm{P}\left(\text { Rejecting } \mathrm{H}_{0} \mid \mathrm{H}_{0} \text { is true }\right)=\alpha$
  • $\text { P(Type II error) }$$=\mathrm{P}\left(\text { Accepting } \mathrm{H}_{0} \mid \mathrm{H}_{0} \text { is false }\right)=\beta$

The level of significance of the test
  • $\alpha = \text{ P(Type I error) } =\mathrm{P}\left(\text { Rejecting } \mathrm{H}_{0} \mid \mathrm{H}_{0} \text { is true }\right)$
One-sided alternative hypothesis:
  • $ \begin{array}{ll} \mathrm{H}_{0}: \theta=\theta_{\mathrm{o}} \quad\text { or } & \mathrm{H}_{0}: \theta=\theta_{\mathrm{o}} \\ \mathrm{H}_{1}: \theta \gt \theta_{\mathrm{o}} & \mathrm{H}_{1}: \theta \lt \theta_{\mathrm{o}} \end{array} $
Two-sided alternative hypothesis:
  • $ \begin{array}{l} \mathrm{H}_{0}: \theta=\theta_{\mathrm{o}}\\ \mathrm{H}_{1}: \theta \neq \theta_{\mathrm{o}} \end{array} $

Important remarks

  • Calculating the test statistic when standard deviation is known and unknown, and when sample size is large and small.

    • $$ \begin{array}{l|ccl} \hline \text{ Test for } & H_{0} & \text { Test statistic } & \text { Use when } \\ \hline \text{ Pop. mean } \mu & \mu=\mu_{0} & z=\frac{\overline{x}-\mu_{0}}{\frac{\sigma}{\sqrt{n}}} & \text{ Normal dist. or } n \gt 30, \sigma \text{ known } \\ \text{ Pop. mean } \mu & \mu=\mu_{0} & t=\frac{\overline{x}-\mu_{0}}{\frac{s}{\sqrt{n}}} & n \lt 30 \text { and/or } \sigma \text { unknown } \\ \text { Pop. prop. } p & p=p_{0} & z=\frac{\hat{p}-p_{0}}{\sqrt{\frac{p_{0}(1-p_{0})}{n}}} & n \hat{p} \geq 10, n(1-\hat{p}) \geq 10\\ \hline \end{array} $$

  • The test statistic used for different hypothesis and its corresponding rejection rule is:

    • $$ \begin{array}{c|c|l} \hline \begin{array}{c}\text { Hypothesis } \\ \text { testing }\end{array} & \text { Test statistic } & \text { Rejection nule } \\ \hline \text { Z-tests } & \text { Z-value } & \begin{array}{l} \text{ If Test statistic } \geq z_{\frac{\alpha}{2}}, \text{ then reject the null hypothesis } H_0\\ \text{ If Test statistic } \leq - z_{\frac{\alpha}{2}}, \text{ then reject the null hypothesis } H_0 \end{array} \\ \hline \text { t-tests } & \text { t-value } & \begin{array}{l} \text{ If Test statistic } \geq t_{(\frac{\alpha}{2},n-1)}, \text{ then reject the null hypothesis } H_0\\ \text{ If Test statistic } \leq - t_{(\frac{\alpha}{2},n-1)}, \text{ then reject the null hypothesis } H_0 \end{array} \\ \hline \end{array} $$

The test procedure:

  • To reject $H_0$ (accepting $H_1$) or to accept $H_0$ (rejecting $H_1$), we need the following steps:
    1. Determining a test statistic (T.S.)
    2. Determining the significance level $\alpha$
      3. $\alpha = 0.01, 0.025, 0.05$, or $0.10$
    3. Determining the rejection region (R.R.) and the acceptance region (A.R.) of $H_0$
      4. R.R. of $H_0$ depends on $H_1$ and $\alpha$
      • $H_1$ determines the direction of the R.R. of $H_0$
      • $\alpha$ determines the size of the R.R. of $H_0$
    4. Decision:
      5. We reject $H_0$ (accept $H_1$) if the value of the T.S. falls in the R.R. of $H_0$, and vice versa.

2.  Single Sample: Tests Concerning a Single Mean (Variance Known):

Suppose that $X_1, X_2, \cdots, X_n$ is a random sample of size $n$ from distribution with mean $\mu$ and known variance $\sigma^2$. We know that:

  • $E\big(\overline{X}\big)=\mu_{\overline{X}}=\mu $ ; $\quad$ $\operatorname{Var}\big(\overline{X}\big)=\sigma_{\bar{X}}^{2}=\frac{\sigma^{2}}{n}$ ; $\quad$ $\overline{X} \sim N\left(\mu, \frac{\sigma}{\sqrt{n}}\right) \Longleftrightarrow Z=\frac{\overline{X}-\mu}{\sigma / \sqrt{n}} \sim \mathrm{N}(0,1)$
  • Let $\mu_0$ be a given known value.
  • Test Procedure:

    • $$ \begin{array}{r | c c c} \hline \text { Hypotheses } & \mathrm{H}_{0}: \mu=\mu_{0} & \mathrm{H}_{0}: \mu=\mu_{0} & \mathrm{H}_{0}: \mu=\mu_{0} \\ & \mathrm{H}_{1}: \mu \neq \mu_{0} & \mathrm{H}_{1}: \mu \gt \mu_{0} & \mathrm{H}_{1}: \mu \lt \mu_{0} \\ \hline \text{ Test Statistic (T.S.) } & & Z=\frac{\overline{X}-\mu_0}{\sigma / \sqrt{n}} \sim \mathrm{N}(0,1) & \\ \hline \text{ R.R. and A.R. of $H_0$ } & & & \\ &&& \\ &&&\\ \hline \text{ Decision: } &\text{ Reject } H_0 \text{ (and accept} & H_1) \text{ at the signification} & \text{level } \alpha \text{ if:} \\ \hline & Z \gt z_{\frac{\alpha}{2}} \text{ or } Z \lt -z_{\frac{\alpha}{2}} & Z \gt z_{\alpha} & Z \lt -z_{\alpha}\\ & \text{ Two-Sided Test } & \text{ One-Sided Test } & \text{ One-Sided Test } \\ \hline \end{array} $$

Example

  1. A random sample of $100$ recorded deaths in the United States during the past year showed an average of $71.8$ years. Assuming a population standard deviation of $8.9$ year, does this seem to indicate that the mean life span today is greater than $70$ years? Use a $0.05$ level of significance.

  2. A manufacturer of sports equipment has developed a new synthetic fishing line that he claims has a mean breaking strength of $8$ kilograms with a standard deviation of $0.5$ kilograms. Test the hypothesis that $\mu =8$ kg against the alternative that $\mu \neq 8$ kg if a random sample of $50$ lines is tested and found to have a mean breaking strength of $7.8$ kg. Use a $0.01$ level of significance.

3. Single Sample: Tests on a Single Mean (Variance Unknown):

Suppose that $X_1, X_2, \cdots, X_n$ is a random sample of size $n$ from normal distribution with mean $\mu$ and unknown variance $\sigma^2$. We know that:

  • $T=\frac{\overline{X}-\mu}{S / \sqrt{n}} \sim \mathrm{t}(\text{df})$ ; $\quad \text{df}=n-1$ ; $\quad S=\sqrt{\sum_{i=1}^{n}\big(X_{i}-\overline{X}\big)^{2} /(n-1)}$
  • Let $\mu_0$ be a given known value.
  • Test Procedure:

    • $$ \begin{array}{r | c c c} \hline \text { Hypotheses } & \mathrm{H}_{0}: \mu=\mu_{0} & \mathrm{H}_{0}: \mu=\mu_{0} & \mathrm{H}_{0}: \mu=\mu_{0} \\ & \mathrm{H}_{1}: \mu \neq \mu_{0} & \mathrm{H}_{1}: \mu \gt \mu_{0} & \mathrm{H}_{1}: \mu \lt \mu_{0} \\ \hline \text{ Test Statistic (T.S.) } & & T=\frac{\overline{X}-\mu_0}{S / \sqrt{n}} \sim \mathrm{t}(\text{n-1}) & \\ \hline \text{ R.R. and A.R. of $H_0$ } & & & \\ &&& \\ &&&\\ \hline \text{ Decision: } &\text{ Reject } H_0 \text{ (and accept } H_1) & \text{ at the signification} & \text{level } \alpha \text{ if:} \\ \hline & T \gt t_{(\frac{\alpha}{2},n-1)} \text{ or } T \lt -t_{(\frac{\alpha}{2},n-1)} & Z \gt t_{(\alpha,n-1)} & Z \lt -t_{(\alpha,n-1)}\\ & \text{ Two-Sided Test } & \text{ One-Sided Test } & \text{ One-Sided Test } \\ \hline \end{array} $$

Example:

ប្រសិនបើសំណាកចៃដន្យមួយ អង្កេតលើខ្នងផ្ទះចំនួន $12$ បានបង្ហាញថា ម៉ាស៊ីនបូមធូលីរបស់ពួកគេប្រើភ្លើងអស់ជាមធ្យមចំនួន $42$ គីឡូវ៉ាត់ម៉ោងក្នុងមួយឆ្នាំ ដោយមានគម្លាតស្តង់ដា $11.9$ គីឡូវ៉ាត់ម៉ោង នោះតើសម្មតិកម្មនេះ បង្ហាញនូវកម្រិតសារៈសំខាន់ (level of significance) ចំនួន $0.05$ ដែលម៉ាស៊ីនបូមធូលីប្រើភ្លើងជាមធ្យម តិចជាង $46$ គីឡូវ៉ាត់ម៉ោងក្នុងមួយឆ្នាំ ដែរឬទេ? សន្មតថាប៉ូពុយឡាស្យុងនៃគីឡូវ៉ាត់ម៉ោង ជារបាយន័រម៉ាល់។


4. Two Samples: Tests on Two Means:

For two independent samples, we have:

  • If \(\sigma_{1}^{2}\) and \(\sigma_{2}^{2}\) are known, then we get: $$ Z=\frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim \mathrm{N}(0,1) $$
  • If \(\sigma_{1}^{2}\) and \(\sigma_{2}^{2}\) are unknown but \(\sigma_{1}^{2}=\sigma_{2}^{2}=\sigma^{2},\) then we have: $$ T=\frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-\left(\mu_{1}-\mu_{2}\right)}{S_{p} \sqrt{\frac{1}{n_{1}}+\frac{1}{n_{2}}}} \sim \mathrm{t}\left(n_{1}+n_{2}-2\right) $$
    • Where the pooled estimate of \(\sigma^{2}\) is $$ S_{p}^{2}=\frac{\left(n_{1}-1\right) S_{1}^{2}+\left(n_{2}-1\right) S_{2}^{2}}{n_{1}+n_{2}-2} $$ The degrees of freedom of $S^2_p$ is df$=n_1+n_2-2$.
  • Now, suppose we need to test the null hypothesis $H_0 : \mu_1 - \mu_2 = 0$

Generally, suppose we need to test $$H_0 : \mu_1 - \mu_2 = d, \quad \text{ (for some specific value } d)$$ Against one of the following alternative hypothesis $$ \mathrm{H}_{1}: \left\{ \begin{array}{l} \mu_{1}-\mu_{2} \neq d \\ \mu_{1}-\mu_{2} \gt d \\ \mu_{1}-\mu_{2} \lt d \end{array}\right. $$

$$ \begin{array}{r | c c c} \hline \text { Hypotheses } & \mathrm{H}_{0}: \mu_1-\mu_2=d & \mathrm{H}_{0}: \mu_1-\mu_2= d& \mathrm{H}_{0}: \mu_1-\mu_2=d \\ & \mathrm{H}_{0}: \mu_1-\mu_2\neq d & \mathrm{H}_{0}: \mu_1-\mu_2\gt d & \mathrm{H}_{0}: \mu_1-\mu_2\lt d \\ \hline \text{ Test Statistic (T.S.) } & Z=\frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-d}{\sqrt{\frac{\sigma_{1}^{2}}{n_{1}}+\frac{\sigma_{2}^{2}}{n_{2}}}} \sim \mathrm{N}(0,1) & \{\text{if } \sigma^2_1 \text{ and } \sigma^2_2 \text{ are known }\}& \\ &\text{ or } T=\frac{\left(\bar{X}_{1}-\bar{X}_{2}\right)-d}{S_p\sqrt{\frac{1}{n_{1}}+\frac{1}{n_{2}}}} \sim \mathrm{t}(n_1+n_2-2) & \{\text{if } \sigma^2_1 = \sigma^2_2 = \sigma^2 \text{ is unknown }\}& \\ \hline \text{ R.R. and A.R. of $H_0$ } & & & \\ &&& \\ &&&\\ &&&\\ \hline \text{ Decision: } &\text{ Reject } H_0 \text{ (and accept } H_1) & \text{ at the signification} & \text{level } \alpha \text{ if:} \\ \hline & \text{ T.S. } \in \text{ R.R. } & \text{ T.S. } \in \text{ R.R. } & \text{ T.S. } \in \text{ R.R. }\\ & \text{ Two-Sided Test } & \text{ One-Sided Test } & \text{ One-Sided Test } \\ \hline \end{array} $$

Example:

An experiment was performed to compare the abrasive wear of were tested by exposing each piece to a machine measuring wear. Ten pieces of material 2 were similarly tested. In each case, the depth of wear was observed. The samples of material 1 gave an average wear of 85 units with a sample standard deviation of 4, while the samples of materials 2 gave an average wear of 81 and a sample standard deviation of 5. Can we conclude at the 0.05 level of significance that the mean abrasive wear of material 1 exceeds that of material 2 by more than 2 units? Assume populations to be approximately normal with equal variances.

4. One Sample: Tests on a Single Proportion:

Recall: For one samples proportion:

  • $p =$ Population proportion of elements of Type $A$ in the population $= \frac{A}{A+B}$
  • $X=$ no.of elements of type A in the sample of size $n$.
  • $\widehat{p} = $ Sample proportion elements of Type $A$ in the sample $=\frac{X}{n}$
  • For large $n$, we have
    • $$ Z=\frac{\widehat{p}-p}{\sqrt{\frac{p q}{n}}}=\frac{X-n p}{\sqrt{n p q}} \sim \mathrm{N}(0,1) ; \quad \text{(Approximately, } q=1−p) $$
  • Test Procedure:

$$ \begin{array}{r | c c c} \hline \text { Hypotheses } & \mathrm{H}_{0}: p=p_{0} & \mathrm{H}_{0}: p=p_{0} & \mathrm{H}_{0}: p=p_{0} \\ & \mathrm{H}_{1}: p \neq p_{0} & \mathrm{H}_{1}: p \gt p_{0} & \mathrm{H}_{1}: p \lt p_{0} \\ \hline \text{ Test Statistic (T.S.) } & Z=\frac{\widehat{p}-p_0}{\sqrt{\frac{p_0 q_0}{n}}} & =\frac{X - n p_0}{\sqrt{n p_0 q_0}} \sim \mathrm{N}(0,1) & (q_0 = 1- p_0)\\ \hline \text{ R.R. and A.R. of $H_0$ } & & & \\ &&& \\ &&&\\ &&&\\ \hline \text{ Decision: } &\text{ Reject } H_0 \text{ (and accept} & H_1) \text{ at the signification} & \text{level } \alpha \text{ if:} \\ \hline & Z \gt z_{\frac{\alpha}{2}} \text{ or } Z \lt -z_{\frac{\alpha}{2}} & Z \gt z_{\alpha} & Z \lt -z_{\alpha}\\ & \text{ Two-Sided Test } & \text{ One-Sided Test } & \text{ One-Sided Test } \\ \hline \end{array} $$

Example:

A builder claims that heat pumps are installed in 70% of all homes being constructed today in the city. Would you agree with this claim if a random survey of new homes in the city shows that 8 out of 15 homes had heat pumps installed? Use a 0.10 level of significance.

5. Two Samples: Tests on Two Proportions

Suppose that we have two populations: $p_1 =$ proportion of the 1-st population and $p_2 =$ proportion of the 2-st population. We are interested in comparing $p_1$ and $p_2$, or equivalently, making inferences about $p_1- p_2$.

  • Test procedure: Suppose we need to test:
    • $$ \begin{array}{l} \mathrm{H}_{0}: p_{1}-p_{2}=0 \\ \text{ Against }\mathrm{H}_{1}: \left\{\begin{array}{l} p_{1}-p_{2} \neq 0 \\ p_{1}-p_{2} \gt 0 \\ p_{1}-p_{2} \lt 0\end{array}\right. \end{array} $$ Note, under $H_0: p_1= p_2= p$, the pooled estimate of the proportion $p$ is: $$ \widehat{p}=\frac{X_{1}+X_{2}}{n_{1}+n_{2}} ; \quad(\hat{q}=1-\hat{p}) $$ The test statistic (T.S.) is $$ Z=\frac{\left(\hat{p}_{1}-\hat{p}_{2}\right)}{\sqrt{\hat{p} \hat{q}\left(\frac{1}{n_{1}}+\frac{1}{n_{2}}\right)}} \sim \mathrm{N}(0,1) $$

រំលឹក :

  • $X \sim \operatorname{Binomial}(n, p)$
  • $\mathrm{E}(\hat{p})=\mathrm{E}\left(\frac{X}{n}\right)=p$
  • $\operatorname{Var}(\hat{p})=\operatorname{Var}\left(\frac{X}{n}\right)=\frac{p q}{n}; \quad q=1-p$
  • For large $n$, we have $$ \begin{aligned} \hat{p} &\sim \mathrm{N}\left(p, \sqrt{\frac{p q}{n}}\right) &\text { (Approximately) } \\ Z&=\frac{\hat{p}-p}{\sqrt{\frac{p q}{n}}} \sim \mathrm{N}(0,1) &\text { (Approximately) } \end{aligned}$$

  • Point Estimation for $p$: A good point estimator for the population proportion $p$ is given by the statistic (sample proportion): $\hat{p} = \frac{X}{n}$
  • Confidence Interval for $p$: For large $n$, an approximate $(1-\alpha)100\%$ confidence interval for $p$ is : $$ \hat{p}-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}} \lt p \lt \hat{p}+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p} \hat{q}}{n}}; \quad \hat{q} = 1 - \hat{p} $$

ឧទាហរណ៍.

In a random sample of $n=500$ families owing television sets in the city of Hamilton, Canada, it was found that $x=340$ subscribed to HBO. Find $95\%$ confidence interval for the actual proportion of families in this city who subscribe to HBO.

5. Two Samples: Estimating the Difference between Two Proportions

Suppose that we have two populations:
  • $p_1 =$ proportion of the 1-st population.
  • $p_2 =$ proportion of the 2-nd population.
  • We are interested in comparing $p_1$ and $p_2$, or equivalently, making inferences about $p_1- p_2$.
  • We independently select a random sample of size $n_1$ from the 1-st population and another random sample of size $n_2$ from the 2-nd population:
  • Let $X_1 =$ no. of elements of type A in the 1-st sample: $X_1\sim \operatorname{Binomial}(n_1, p_1)$
  • Let $X_2 =$ no. of elements of type A in the 2-nd sample: $X_2\sim \operatorname{Binomial}(n_2, p_2)$
  • $\hat{p}_{1}=\frac{X_{1}}{n_{1}}=$ proportion of the 1-st sample
  • $\hat{p}_{2}=\frac{X_{2}}{n_{2}}=$ proportion of the 2-st sample

តាមសម្មតិកម្មខាងលើ គេបានកំណត់បានលទ្ធផល ដូចតទៅ៖

  1. $E\left(\hat{p}_{1}-\hat{p}_{2}\right)=p_{1}-p_{2}$
  2. $\operatorname{Var}\left(\hat{p}_{1}-\hat{p}_{2}\right)=\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}} ;\quad q_{1}=1-p_{1}, q_{2}=1-p_{2}$
  3. For large $n_{1}$ and $n_{2}$, we have \begin{eqnarray*} \hat{p}_{1}-\hat{p}_{2} &\sim & N\left(p_{1}-p_{2}, \sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}\right); \quad \text{ (Approximately) } \\ Z &=&\frac{\left(\hat{p}_{1}-\hat{p}_{2}\right)-\left(p_{1}-p_{2}\right)}{\sqrt{\frac{p_{1} q_{1}}{n_{1}}+\frac{p_{2} q_{2}}{n_{2}}}} \sim \mathrm{N}(0,1); \quad \text { (Approximately) } \end{eqnarray*}

  • Point Estimation of $p_1 - p_2$. A good point estimator for the difference between the two proportions, $p_1- p_2$, is given by the statistic: $$ \hat{p}_{1}-\hat{p}_{2}=\frac{X_{1}}{n_{1}}-\frac{X_{2}}{n_{2}} $$
  • Confidence Interval for $p_1 - p_2$. For large \(n_{1}\) and \(n_{2},\) an approximate \((1-\alpha) 100 \%\) confidence interval for \(p_{1}-p_{2}\) is : $$ \left(\hat{p}_{1}-\hat{p}_{2}\right)-z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} \lt p_{1}-p_{2} \lt \left(\hat{p}_{1}-\hat{p}_{2}\right)+z_{\frac{\alpha}{2}} \sqrt{\frac{\hat{p}_{1} \hat{q}_{1}}{n_{1}}+\frac{\hat{p}_{2} \hat{q}_{2}}{n_{2}}} $$

ឧទាហរណ៍.

A certain change in a process for manufacture of component parts is being considered. Samples are taken using both existing and the new procedure to determine if the new process results in an improvement. If $75$ of $1500$ items from the existing procedure were found to be defective and $80$ of $2000$ items from the new procedure were found to be defective, find $90\%$ confidence interval for the true difference in the fraction of defectives between the existing and the new process.

លំហាត់អនុវត្តន៍
