Set theory is the branch of mathematical logic that studies sets, which can be informally described as collections of objects. Although objects of any kind can be collected into a set, set theory, as a branch of mathematics, is mostly concerned with those that are relevant to mathematics as a whole.

The modern study of set theory was initiated by the German mathematicians Richard Dedekind and Georg Cantor in the 1870s. In particular, Georg Cantor is commonly considered the founder of set theory. The non-formalized systems investigated during this early stage go under the name of naive set theory. After the discovery of paradoxes within naive set theory (such as Russell's paradox, Cantor's paradox and Burali-Forti paradox) various axiomatic systems were proposed in the early twentieth century, of which Zermelo–Fraenkel set theory (with or without the axiom of choice) is still the best-known and most studied.

Mathematical topics typically emerge and evolve through interactions among many researchers. Set theory, however, was founded by a single paper in 1874 by Georg Cantor: "On a Property of the Collection of All Real Algebraic Numbers". Since the 5th century BC, beginning with Greek mathematician Zeno of Elea in the West and early Indian mathematicians in the East, mathematicians had struggled with the concept of infinity. Especially notable is the work of Bernard Bolzano in the first half of the 19th century. Modern understanding of infinity began in 1870–1874, and was motivated by Cantor's work in real analysis. An 1872 meeting between Cantor and Richard Dedekind influenced Cantor's thinking, and culminated in Cantor's 1874 paper.

  Basic Concepts

A set is a collection of elements. A set is defined entirely by the elements that it contains. An element can be anything, including another set. You will notice that this is not a precise mathematical definition. Instead, it is an intuitive description of what the word ‘set’ is supposed to mean: any time you have a bunch of entities and you consider them as a unit, you have a set. Mathematically, sets are really defined by the operations that can be performed on them. These operations model things that can be done with collections of objects in the real world. These operations are the subject of the branch of mathematics known as set theory.

The most basic operation in set theory is forming a set from a given list of specific entities. The set that is formed in this way is denoted by enclosing the list of entities between a left brace, ‘$\big\{$’, and a right brace, ‘$\big\}$’. The entities in the list are separated by commas. For example, the set denoted by $$\big\{ 17, \pi , \text{ Phnom Penh City}, \text{ King Willem-Alexander }, \text{ Euromast} \}$$ is the set that contains the entities $17, \pi$ , Phnom Penh City, King Willem-Alexander, and Euromast. These entities are the elements of the set. Since we assume that a set is completely defined by the elements that it contains, the set is well-defined. Of course, we still haven’t said what it means to be an ‘entity’. Something as definite as ‘Phnom Penh City’ should qualify, except that it doesn’t seem like Phnom Penh City really belongs in the world of mathematics. The problem is that mathematics is supposed to be its own self-contained world, but it is supposed to model the real world. When we use mathematics to model the real world, we admit entities such as Phnom Penh City and even Euromast. But when we are doing mathematics per se, we’ll generally stick to obviously mathematical entities such as the integer $17$ or the real number $\pi$ . We will also use letters such as a and b to refer to entities. For example, when saying something like “Let $A$ be the set $\{a, b, c\}$”, it means $a, b$, and $c$ to be particular, but unspecified, entities.

Note:

  • It’s important to understand that a set is defined by the elements that it contains, and not by the order in which those elements might be listed.
  • A set can only contain one copy of a given element, even if the notation that specifies the set lists the element twice. Note in particular that it’s incorrect to say that the set $\{a, b, a, a, b, c, a\}$ contains seven elements, since some of the elements in the list are identical.
  • The notation $\{a, b, c\}$ can lead to some confusion, since it might not be clear whether the letters $a, b$, and $c$ are assumed to refer to three different entities. So, a mathematician generally makes assumption that the set denoted by $\{a, b, c\}$ meaning that different letters refer to different entities.

    • Sets and subsets

    (well-defined collection)

    A well-defined collection of objects (បណ្ដុំនៃវត្ថុដែលមានសញ្ញាណកំណត់ច្បាស់លាស់) is called a set. If $A$ is a set, then the objects in $A$ are called elements/members/entities of $A$. We write $a \in A$, read '$a$ belongs to $A$', when a is an object in $A$. We write $a\notin A$ to say that '$a$ does not belong to $A$'.

    Sets are usually denoted by uppercase letters, such as $A, B, X, Y$, etc. and the elements of sets are denoted by lowercase letters, such as $a, b, x, y$, etc.


    1. A collection of all tall persons in Cambodia is not a set.
    2. The collection of all prime numbers is a set.
    3. The collection of all positive integers, which are not prime, is a set.
    (Empty set)

    A collection having no objects is clearly a set and is called the empty set or null set (សំណុំទទេ) and is denoted by the symbol $\varnothing$ or $\{ \}$.

    Consider the following sets:
    • $A =\{x \mid\, x \text{ is an even integer and } 2 \lt x \lt 4\}$
    • $B = \{x :\, x \text{ is an integer and } x^2 + 2 = 0\}$
    (Equality and subset)

    Let $A$ and $B$ be two sets. Then, we say that

    1. $A$ is equal to $B$ and express this by $A=B$ if, for any object $x$, $$ x \in A \Longleftrightarrow x \in B $$
    2. $A$ is a subset of $B$ (or $A$ is contained in $B$ ) and express this by $A \subseteq B$ if, for any object $x$, $$ x \in A \Longrightarrow x \in B $$

    For any two sets $A$ and $B$, clearly $A=B$ if and only if $A \subseteq B$ and $B \subseteq A$. Whenever we are required to prove that two given sets $A$ and $B$ are equal, we usually prove that $A \subseteq B$ and $B \subseteq A$, that is, for any object $x$, $$ x \in A \Rightarrow x \in B \quad \text { and } \quad x \in B \Rightarrow x \in A . $$ Sometimes, we say that a set $A$ is smaller than $B$ (or $B$ is larger than $A$ ) if $A$ is a subset of $B . A$ is said to be a proper subset of $B$ and write $A \subset B$ if $A \subseteq B$ and $A \neq B$. Also, instead of writing $A \subseteq B$ or $\mathrm{A} \subset B$, some times we write $B \supseteq A$ or $B \supset A$ and say that $B$ contains $A$ (or $B$ is a superset of $A$ ) or $B$ properly contains $A$, respectively. We write $A \nsubseteq B$ if $A$ is not a subset of $B$.

    (Union and Intersection)

    Let \(A\) and \(B\) be subsets of some universal set \(U\). The intersection of \(A\) and \(B\), written \(A \cap B\) and read “\(A\) intersect \(B\),” is the set of all elements that are in both \(A\) and \(B\). That is, \[A \cap B = \{x \in U \, | \, x \in A \text{ and } x \in B\}.\] The union of \(A\) and \(B\), written \(A \cup B\) and read “\(A\) union \(B\),” is the set of all elements that are in \(A\) or in \(B\). That is, \[A \cup B = \{x \in U \, | \, x \in A \text{ or } x \in B\}.\]

    (Difference and Complement)

    Let \(A\) and \(B\) be subsets of some universal set \(U\). The set difference of \(A\) and \(B\), or relative complement of \(B\) with respect to \(A\), written \(A -B\) and read “\(A\) minus \(B\)” or “the complement of \(B\) with respect to \(A\),” is the set of all elements in \(A\) that are not in \(B\). That is, \[A \setminus B = \{x \in U \, | \, x \in A \text{ and } x \notin B\}.\] The complement of the set \(A\), written \(A^c\) and read “the complement of \(A\),” is the set of all elements of \(U\) that are not in \(A\). That is, \[A^c = \{x \in U \, | \, x \notin A\}.\]

      The Power Set of a Set (សំណុំរងដោយផ្នែកៗ)

    (power set=សំណុំស្វ័យគុណ)

    If \(A\) is a subset of a universal set \(U\), then the set whose members are all the subsets of \(A\) is called the power set of \(A\). We denote the power set of \(A\) by \(\mathcal{P}(A)\). Symbolically, we write \[\mathcal{P}(A) = \bigl\{X \subseteq U \, | \, X \subseteq A\bigr\}\] That is, \(X \in \mathcal{P}(A)\) if and only if \(X \subseteq A\).

    When dealing with the power set of \(A\), we must always remember that \(\varnothing \subseteq A\) and \(A \subseteq A\). Now, let \(A = \{a, b\}\), then the subsets of \(A\) are \[\emptyset, \{a\}, \{b\}, \{a,b\}\] We can write the powerset of $A$ as \[\mathcal{P}(A) = \bigg\{\emptyset, \{a\}, \{b\}, \{a,b\}\bigg\}\]

    How about \(B = \{1, 2, 3\}\). Find $\mathcal P(B)$?

    Show it!

    \[ \mathcal{P}(B) = \bigg\{\varnothing, \{1\}, \{2\}, \{1,2\}, \{3\}, \{1, 3\}, \{2, 3\}, B\bigg\} \]

    Theorem 1: (total elements of powerset)

    Let \(n\) be a nonnegative integer and let \(A\) be a subset of some universal set. If the set \(A\) has \(n\) elements, then the set \(A\) has \(2^n\) subsets. That is, \(\mathcal{P}(A)\) has \(2^n\) elements.

    NOTE: Note that the power set $\mathcal{P}(A)$ of any set $A$ is always nonempty, since the empty set $\varnothing$ is a subset of every set $A$. In fact, if $A$ is the empty set $\varnothing$, then $$ \mathcal{P}(\varnothing)=\{\varnothing\}, $$ a set consisting of only one element. It can be easily proved that, for any nonnegative integer $n$, a set $A$ has exactly $n$ elements if and only if the power set $\mathcal{P}(A)$ has exactly $2^{n}$ elements.

    (Class/family of set)

    A set whose element are sets is called a class of sets (ថ្នាក់នៃសំណុ) or family of sets.

    Class of sets will be usually denoted by script letters, such as $\mathscr{A}, \mathscr{B}, \mathscr{C}$, etc. For any set $A$, the power set $\mathcal{P}(A)$ is a class of sets. A class $\mathscr{C}$ of sets is called an indexed class if there exists a set $I$ such that, for each $i \in I$, there is a unique member $A_{i}$ in $\mathscr{C}$ associated with $i$ and the class $\mathscr{C}$ is equal to the class of all $A_{i}$, $i \in I$; in this case, we write $$ \mathscr{C}=\left\{A_{i}: i \in I\right\} \quad \text { or } \quad \mathscr{C}=\left\{A_{i}\right\}_{i \in I} $$ and $I$ is called the index set.

    For any positive integer $n$, let $$ A_{n}=\left\{x\mid\; x \text { is a real number and } 0 \leq x \leq \frac{1}{n}\right\} $$ Then, $\left\{A_{n}\right\}_{n \in \mathbb{Z}^{+}}$ is an indexed class of sets and the set $\mathbb{Z}^{+}$ of positive integers is the index set.

    The Cardinality of a Finite Set

    In our discussion of the power set, we were concerned with the number of elements in a set. In fact, the number of elements in a finite set is a distinguishing characteristic of the set, so we give it the following name.

    (cardinality)

    The number of elements in a finite set \(A\) is called the cardinality of \(A\) and is denoted by $\mathbf{card}(A)$ or $n(A)$ or $|A|$ or $\#A$.

    1. $\card(\varnothing) = 0$
    2. $\card\big(\{a, b\}\big) = 2$
    3. $\card(\mathcal{P}(\{a, b\})) = 4$

    Operations on a class of subsets

    For any indexed class of sets $\left\{A_{i}\right\}_{i \in I}$, we define the set as $$ \bigcap_{i \in I} A_{i}=\left\{a \mid\; a \in A_{i} \;\;\text { for all } i \in I\right\} \text {. } $$ This set is called the set intersection of $A_{i}$ 's, $i \in I$. In particular, if $A_{1}, A_{2}, \ldots, A_{n}$ are sets, we define $$ \bigcap_{i=1}^{n} A_{i}=\left\{a\mid\; a \in A_{i} \;\;\text { for } i=1,2, \ldots, n\right\} $$ and is also denoted by $A_{1} \cap A_{2} \cap \ldots \cap A_{n}$.

    Remind: For any sets $A$ and $B$, we have $$ A \cap B=\{x: x \in A \text { and } x \in B\} . $$ Two sets $A$ and $B$ are said to be disjoint if $A \cap B=\varnothing$, that is, there are no common elements of $A$ and $B$.

    For any indexed class $\left\{A_{i}\right\}_{i \in I}$ of sets, we define the set as $$ \bigcup_{i \in I} A_{i}=\left\{a \mid\, a \in A_{i} \;\;\text { for some } i \in I\right\} \text {. } $$ This set is called the set union of $A_{i}$ 's, $i \in I$. In particular, for any sets $A_{1}, A_{2}$, $\ldots, A_{n}$, we define $$ \bigcup_{i=1}^{n} A_{i}=\left\{a\mid\, a \in A_{i}\;\; \text { for some } 1 \leq i \leq n\right\} $$ and this is also denoted by $A_{1} \cup A_{2} \cup \ldots \cup A_{n}$.

    Remind: For any sets $A$ and $B$, we have $$ A \cup B=\{x: x \in A \text { or } x \in B\} $$

    For any positive integer $n$, let $$ A_{n}=\left\{x: x \text { is a real number and } 0 \leq x \leq \frac{1}{n}\right\} . $$ Then, we have $A_{n} \supset A_{n+1}$ for any $n$ and therefore $$ [0,1]=A_{1} \supset A_{2} \supset A_{3} \supset \ldots \supset A_{n} \supset A_{n+1} \supset \ldots $$ So, find $\bigcap_{n \in \mathbb{Z}^{+}} A_{n}=?$ and $\bigcup_{n \in \mathbb{Z}^{+}} A_{n}=?$

    Theorem 3: (De Morgan's Duality Laws)

    For any universal sets \(U\) and let \(\mathcal{A} = \{A_{i} | i \in I\}\) be an indexed set family, where \(A_{i}\subseteq U\) \((i \in I)\), the following are true:

    1. \(\displaystyle U \setminus \bigcup_{i} A_{i} = \bigcap_{i}(U \setminus A_i)\)
    2. \(\displaystyle U \setminus \bigcap_{i}A_{i} = \bigcup_{i}(U \setminus A_{i})\)
    We may write \(\overline{\displaystyle\bigcup_i A_{i}}\) for \(\displaystyle U \setminus \bigcup_i A_{i}\).

    Show Proof!

    1. Let $A$ and $B$ be sets. If $A \cap B=\varnothing$, then we say that $A$ and $B$ are disjoint (ដាច់គ្នា).
    2. If $\left\{A_d\right\}_{d \in D}$ is a collection of sets, and every pair of sets in the collection is disjoint, then we say that the sets in the collection are mutually disjoint (ដាច់គ្នាពីរៗ).
