គោលបំណងនៅក្នុងជំពូកនេះគឺ៖

Another way to obtain a new set from two given sets \(A\) and \(B\) is to form ordered pairs. An ordered pair \((x,y)\) consists of two values \(x\) and \(y\). Their order of appearance is important, so we call them first and second elements respectively. Consequently, \((a,b)\neq (b,a)\) unless \(a=b\). In general, \((a,b)=(c,d)\) if and only if \(a=c\) and \(b=d\).

  និយមន័យ

Note:

  • It’s important to understand that a set is defined by the elements that it contains, and not by the order in which those elements might be listed.
  • A set can only contain one copy of a given element, even if the notation that specifies the set lists the element twice. Note in particular that it’s incorrect to say that the set $\{a, b, a, a, b, c, a\}$ contains seven elements, since some of the elements in the list are identical.
  • The notation $\{a, b, c\}$ can lead to some confusion, since it might not be clear whether the letters $a, b$, and $c$ are assumed to refer to three different entities. So, a mathematician generally makes assumption that the set denoted by $\{a, b, c\}$ meaning that different letters refer to different entities.

    (Cartesian Product)

    The Cartesian product of \(A\) and \(B\) is the set \[ A \times B = \Big\{ (a,b) \mid a \in A \wedge b \in B \Big\} \]


    Notation in mathematics is often developed for good reason. In this case, a few examples will make clear why the symbol \(\times\) is used for Cartesian products.
    1. Let \(A = \{1, 2, 3\}\) and \(B = \{4, 5\}\text{.}\) Then \(A \times B = \{(1, 4), (1, 5), (2, 4), (2, 5), (3, 4), (3, 5)\}\text{.}\) Note that \(|A \times B| = 6 = \lvert A \rvert \times \lvert B \rvert \text{.}\)
    2. \(A \times A = \{(1, 1), (1, 2), (1, 3), (2, 1), (2, 2), (2, 3), (3, 1), (3, 2), (3, 3)\}\text{.}\) Note that \(|A \times A| = 9 = {\lvert A \rvert}^2\text{.}\)

    These two examples illustrate the general rule that if \(A\) and \(B\) are finite sets, then \(\lvert A \times B \rvert = \lvert A \rvert \times \lvert B \rvert \text{.}\)
    We can define the Cartesian product of three (or more) sets similarly. For example, \[A \times B \times C = \Big\{(a, b, c):a \in A, b \in B, c \in C\Big\}\]

    (Empty set)

    A collection having no objects is clearly a set and is called the empty set or null set (សំណុំទទេ) and is denoted by the symbol $\varnothing$ or $\{ \}$.

    Consider the following sets:
    • $A =\{x \mid\, x \text{ is an even integer and } 2 \lt x \lt 4\}$
    • $B = \{x :\, x \text{ is an integer and } x^2 + 2 = 0\}$
    (Equality and subset)

    Let $A$ and $B$ be two sets. Then, we say that

    1. $A$ is equal to $B$ and express this by $A=B$ if, for any object $x$, $$ x \in A \Longleftrightarrow x \in B $$
    2. $A$ is a subset of $B$ (or $A$ is contained in $B$ ) and express this by $A \subseteq B$ if, for any object $x$, $$ x \in A \Longrightarrow x \in B $$

    For any two sets $A$ and $B$, clearly $A=B$ if and only if $A \subseteq B$ and $B \subseteq A$. Whenever we are required to prove that two given sets $A$ and $B$ are equal, we usually prove that $A \subseteq B$ and $B \subseteq A$, that is, for any object $x$, $$ x \in A \Rightarrow x \in B \quad \text { and } \quad x \in B \Rightarrow x \in A . $$ Sometimes, we say that a set $A$ is smaller than $B$ (or $B$ is larger than $A$ ) if $A$ is a subset of $B . A$ is said to be a proper subset of $B$ and write $A \subset B$ if $A \subseteq B$ and $A \neq B$. Also, instead of writing $A \subseteq B$ or $\mathrm{A} \subset B$, some times we write $B \supseteq A$ or $B \supset A$ and say that $B$ contains $A$ (or $B$ is a superset of $A$ ) or $B$ properly contains $A$, respectively. We write $A \nsubseteq B$ if $A$ is not a subset of $B$.
